# RestauranteUniversipratico_Avisos

Orquestração do front e back da tela de avisos do RestauranteUniversipratico usando Docker

Certificando-se que o Docker esteja instalado na máquina, basta rodar:

```shell
docker compose up --build
```

Para acessar a aplicação (frontend):

- http://localhost:8080/

A API (backend) estará em:

- http://localhost:8081/avisos
